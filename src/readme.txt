Cómo usar este código.

Paso 1: Instalar nodejs: https://nodejs.org

Paso 2: Instalar visual studio code: https://code.visualstudio.com/download

Paso 3: Instalar librerías del proyecto.
La instalación, como es habitual en nodejs, se realiza a partir de la consola y desde la carpeta en la que está ubicada el proyecto.
Basta con ejecutar el comando npm install ( Importante estar en la carpeta del proyecto)

Paso 4: Ejecución de los programas de consola.
Hay 9 programas de consola. Para ejecutar cualquiera de ellos se debe estar en la carpeta raíz del proyecto y ejecutar "npm run ejemplo#", siendo # el número del programa a ejecutar 1<=n<=9

Paso 5: Ejecución de ejemplos html (ejemplo 10 y 11)
En estos casos se necesita un servidor html. Lo más simple es instalar en visual studio code la extensión "live server".
Esta extensión nos provee de un servidor web que, además, se actualiza automáticamente cuando el código se modifica.
PAra lanzarlo solo hay que situarse en un fichero html y, con el botón derecho del ratón, abrir el menú contextual de Visual Studio Code y elegir la opción "Abrir con Live Serve".
